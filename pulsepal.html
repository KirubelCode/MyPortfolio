Learn More About PulsePal

Overview

PulsePal is an Android fitness app that connects wirelessly to a Movesense wearable sensor to display real-time heart rate, steps, pace, distance, and calorie metrics during exercise sessions. It supports goal setting, post-workout analysis, and cloud sync via a PHP/MySQL backend.

Key Features

  • Live heart rate, steps, distance, pace, and calorie tracking.
  • Set up to 3 performance goals per session and track progress visually.
  • Post-workout summary with charts and in-session goal evaluation.
  • Reliable background operation.
  • Minimal setup, beginner-friendly design.

Architecture

  • Movesense BLE sensor streams data to app using MDS service.
  • SensorDataManager handles RxJava observable streams.
  • Retrofit sends data to a PHP backend every 5 seconds.
  • Session data stored in MySQL and retrieved via REST APIs.

Technology Stack

  • Android SDK (Java), MPAndroidChart, Material Components
  • Movesense SDK (MDS), RxJava, Retrofit, Gson
  • Backend: PHP, MySQL (Hostinger shared hosting)

Use Cases

  • Start, pause, resume and stop workout sessions.
  • Track and visualise heart rate, calories, pace and distance over time.
  • Store data on cloud for cross-device access and comparison.

Challenges & Lessons

  • Improved data accuracy by switching to chest-mounted sensors.
  • Managed BLE concurrency and background session drift.
  • Used SharedPreferences for resilience after crashes or restarts.
  • Reduced data stream frequency to concurrently retrieve the data streams from the sensor.
