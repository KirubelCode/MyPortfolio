Overview

This project explores the development of spam email classifiers using various machine learning models including Na誰ve Bayes, SVM, K-NN, MLP, and K-Means Clustering. It utilises the Enron Email Dataset and follows the CRISP-DM methodology.

Key Features

  • Rule-based spam labeling strategy using sender domains and keyword filters.
  • Text preprocessing with TF-IDF vectorization, stopword removal, and bigrams.
  • Comparison of Na誰ve Bayes, SVM, KNN, and MLP classifiers.
  • Unsupervised K-Means clustering with binary spam classification rules.

Technologies Used

  • Python, Pandas, Scikit-learn, Matplotlib
  • ML Models: Na誰ve Bayes, SVM, k-NN, MLPClassifier, K-Means
  • Text Features: TF-IDF, Bigrams, Regex cleaning

Model Evaluation Highlights

  • Na誰ve Bayes: 83.91% (improved to 95.53%)
  • SVM: 89.77% (improved to 96.79%)
  • KNN: 82.04% (improved to 96.06%)
  • MLP: Strong accuracy, especially in non-spam detection (Precision: 0.98, Recall: 0.99)
  • K-Means: Used for pattern discovery and label inference

Future Improvements

  • Use embeddings instead of TF-IDF for richer representations.
  • Use a pre-labeled email dataset.
